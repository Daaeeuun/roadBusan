# RoadBusan

## 🌟 프로젝트 소개
**RoadBusan**은 부산의 해안길, 숲길, 강변길, 도심길을 아우르는 총 21개 갈맷길 코스를 소개하며 함께 주변 맛집 및 관광지를 제공하는 서비스입니다.
산책을 하면서 맛있는 음식도 즐기고 새로운 명소도 발견할 수 있는 색다른 경험을 제공하는 것이 목표입니다. 

---

## 🔧 기술 스택
- **Frontend**:
  - React.js
  - TypeScript
  - Styled-Components
- **Backend**:
  - Node.js
  - Express
- **API 및 데이터**:
  - OpenWeatherMap API
  - 교통 데이터 API (예: 부산시 교통정보 API)
- **배포**:
  - GitHub Pages
